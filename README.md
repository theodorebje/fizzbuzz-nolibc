# FizzBuzz Without Libc

This is a [C](https://en.wikipedia.org/wiki/C_(programming_language)) port of the [FizzBuzz](https://en.wikipedia.org/wiki/Fizz_buzz) programming challenge written without the use of [`libc`](https://en.wikipedia.org/wiki/C_standard_library). It targets the [`x86_64`](https://en.wikipedia.org/wiki/X86-64) architecture.
